# fuber

This is a taxi booking service app. It has apis to book a taxi at your location for a specified time

## Steps to run
1) npm i
2) npm run start

#### npm run test 
#### npm run lint 

### Available apis

1) /bookTaxiRide 
  input - latitude, longitude, color(opt)

2) /endTaxiRide
  input - latitude, longitude, number(taxi)

3) /getAllAvailableTaxis
